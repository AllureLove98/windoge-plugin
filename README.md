# windoge-plugin
自用Yunzai-bot插件，多为修改适配其他大佬的插件功能


### 安装方法
```
git clone https://github.com/gxy12345/windoge-plugin.git ./plugins/windoge-plugin/
```


### 当前功能
1. #便签 与yunzai自带的体力区分，使用xiaoyao-cvs-plugin中的体力面板展示
2. #便签设置 xiaoyao-cvs-plugin中的相应设置
3. #便签背景图(强制)更新 
4. 待更新...
